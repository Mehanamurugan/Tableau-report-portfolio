# ✈️ Aviation Accident Analysis Dashboard

## 📌 Project Overview
This Tableau project analyzes aviation accident data to identify safety risks, operational patterns, and accident distribution trends across countries, aircraft types, and airports.

---

## 🎯 Objective
To perform data analysis on aviation accident records and extract actionable insights using interactive Tableau dashboards.

---

## 🗂️ Dataset Information
- Total Records: 150,000+ aviation accident records  
- Key Fields:
  - Country
  - Airport
  - Aircraft Type
  - Flight Purpose
  - Damage Severity

---

## 📊 Key Insights
- 97.6% of accidents occurred in the United States  
- Aircraft type analysis highlights high-risk categories  
- Airport-level analysis identifies accident-prone locations  
- Flight purpose segmentation reveals operational risk patterns  

---

## 📈 Dashboard Features
- KPI metrics (Total accidents, distribution analysis)
- Aircraft category breakdown
- Airport-wise accident analysis
- Interactive filters:
  - Country
  - Airport
  - Aircraft Type
  - Flight Purpose
  - Damage Severity

---
