# 📊 Customer Churn Analysis Dashboard

## 📌 Project Overview
This Tableau project analyzes customer churn behavior to identify key factors affecting customer retention and revenue loss.

---

## 🎯 Objective
To understand why customers leave and identify patterns that impact churn rate using interactive Tableau dashboards.

---

## 🗂️ Dataset Information
- Total Records: 7,000+ customer records  
- Key Fields:
  - Customer ID
  - Contract Type
  - Payment Method
  - Monthly Charges
  - Tenure
  - Churn Status

---

## 📊 Key Insights
- Overall churn rate: **26.54%**  
- Revenue loss: **₹139,131**  
- Month-to-month contracts have highest churn: **42.71%**  
- Electronic check users show higher churn risk  
- Customers with low tenure (0–5 months) have highest churn rate (**55%**)  

---

## 📈 Dashboard Features
- KPI metrics (Churn rate, revenue loss)
- Contract-wise churn analysis
- Payment method analysis
- Tenure-based churn trends
- Customer segmentation filters

---
